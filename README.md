# JOGO DA VELHA ONLINE

Um jogo da velha multiplayer em tempo real desenvolvido com JavaScript, WebSocket e Tailwind.

## Link para acesso online
https://jogo-da-velha-0l41.onrender.com/

## Tecnologias usadas
Node.js
WebSocket
Tailwind
Jest

## Funcionalidades
-Criação de salas
-Usar código para entrar em partida
-Multiplayer em tempo real
-Controle de turnos
-Vitória e empate
-Sistema de revanche
-Detecta desconexão de jogador

## Testes

O projeto possui testes automatizados utilizando Jest.

Os testes validam:

-Estado inicial da partida
-Entrada de jogadores
-Alternância correta de turnos
-Validação de jogadas
-Validação de jogadas inválidas
-Bloqueio de jogadas inválidas
-Detecção de vitória
-Encerramento de partida
-Reinicialização da partida para jogar novamente

## Executar teste

npm test
